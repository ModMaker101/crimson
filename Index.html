About Crimson

Crimson is a revolutionary new programming language built from the ground up to address the challenges of modern software development. It combines the best aspects of established languages with innovative features to provide an unparalleled development experience.

Our focus is on creating a language that is not only powerful and fast but also intuitive and enjoyable to write. With a clean syntax and robust standard library, Crimson empowers developers to build complex applications with ease and confidence.

Crimson Code Examples

Hello World!


print("Hello, World!");

A classic start. The `print` statement outputs text to the console.

Variable Declaration and Type Inference


var message: string = "Crimson is awesome!";
var year = 2025; // Type inferred as int
print($"{message} in {year}");

Variables can be declared with explicit types or allow type inference. String interpolation makes output easy.

Function Definition and Call


func add(a: int, b: int): int {
    return a + b;
}

func main() {
    var result = add(5, 3);
    print($"5 + 3 = {result}");
}

main(); // Call the main function

Define functions with type-annotated parameters and return types. Functions are first-class citizens.

Conditional Logic (If-Else)


var temperature = 25;

if (temperature > 30) {
    print("It's hot!");
} else if (temperature > 20) {
    print("It's warm.");
} else {
    print("It's cool.");
}

Standard `if-else if-else` constructs for branching logic.

Loops (For Loop)


for (var i = 0; i < 5; i = i + 1) {
    print($"Iteration: {i}");
}

Iterate through a range with a `for` loop.

Class and Object (Basic)


class Dog {
    var name: string;

    func constructor(n: string) {
        this.name = n;
    }

    func bark() {
        print($"{this.name} says Woof!");
    }
}

var myDog = new Dog("Buddy");
myDog.bark();

Demonstrates basic class definition, constructor, fields, and methods.
